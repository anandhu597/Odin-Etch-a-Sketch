# Etch-a-Sketch

A browser-based drawing toy built with vanilla JavaScript and CSS.

## How to play

- Move your mouse over the grid to draw
- Each square gets a random color on first touch
- Hover the same square multiple times to darken it
- Hit **RESIZE** to pick a new grid size (up to 100x100)

## Built with

- HTML
- CSS (Flexbox)
- Vanilla JavaScript (DOM manipulation, event delegation)

## Features

- 16x16 default grid
- Random RGB colors per square
- Progressive darkening over 10 interactions
- Dynamic grid resizing
